12.2.10 _sbrk_r—Reentrant version of sbrk

Synopsis 

     #include <reent.h>
     void *_sbrk_r(struct _reent *ptr, ptrdiff_t incr);

Description
This is a reentrant version of sbrk. It takes a pointer to the global data block, which holds errno.

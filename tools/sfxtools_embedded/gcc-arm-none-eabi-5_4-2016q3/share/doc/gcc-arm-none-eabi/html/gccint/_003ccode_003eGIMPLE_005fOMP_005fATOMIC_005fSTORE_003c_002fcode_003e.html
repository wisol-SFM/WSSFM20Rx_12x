11.8.13 GIMPLE_OMP_ATOMIC_STORE

Build a GIMPLE_OMP_ATOMIC_STORE statement. VAL is the value to be stored.

— GIMPLE function: tree gimple_omp_atomic_store_val ( const gomp_atomic_store *g)

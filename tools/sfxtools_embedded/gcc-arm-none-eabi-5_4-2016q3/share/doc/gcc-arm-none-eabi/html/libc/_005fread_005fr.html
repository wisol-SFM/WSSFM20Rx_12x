12.2.9 _read_r—Reentrant version of read

Synopsis 

     #include <reent.h>
     _ssize_t _read_r(struct _reent *ptr,
         int fd, void *buf, size_t cnt);

Description
This is a reentrant version of read. It takes a pointer to the global data block, which holds errno.

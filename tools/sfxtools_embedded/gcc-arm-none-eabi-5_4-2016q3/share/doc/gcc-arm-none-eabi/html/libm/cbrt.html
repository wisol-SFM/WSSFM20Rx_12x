1.11 cbrt, cbrtf—cube root

Synopsis 

     #include <math.h>
     double cbrt(double x);
     float  cbrtf(float x);

Description
cbrt computes the cube root of the argument.
Returns
The cube root is returned.
Portability
cbrt is in System V release 4. cbrtf is an extension.

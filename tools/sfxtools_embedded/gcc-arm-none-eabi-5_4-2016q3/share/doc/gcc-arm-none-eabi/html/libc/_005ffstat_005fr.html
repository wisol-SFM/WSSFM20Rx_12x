12.2.5 _fstat_r—Reentrant version of fstat

Synopsis 

     #include <reent.h>
     int _fstat_r(struct _reent *ptr,
         int fd, struct stat *pstat);

Description
This is a reentrant version of fstat. It takes a pointer to the global data block, which holds errno.
